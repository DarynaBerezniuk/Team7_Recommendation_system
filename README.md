# Комп'ютерний проєкт на тему "рекомендаційна система". Додаток для знайомств
Користувач отримує випадкову людину з певним набором інтересів. На основі кожного опитування система підбирає наступну людину і так до 10 людей. У кінці система виводить топ підходящих по інтересах людей, котрі ще не були опитані. У роботі буде використаний алгоритм PageRank.

## Розподіл праці
Основний алгоритм — Софія Садовенко та Іван Антоненко;
Тестування і докер — Тарас Коновал;
CLI та візуалізація — Дарина Березнюк.
